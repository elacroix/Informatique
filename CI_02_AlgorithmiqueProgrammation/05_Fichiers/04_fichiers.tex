Fichiers
-	Séparation des données et du programme dans des fichiers séparés.
-	Un fichier est une ressource critique : les opérations sur les fichiers passent par le système.
-	notion de chemin d’accès,
-	lecture et écriture de données numériques ou de type chaîne de caractères depuis ou vers un fichier.
3 étapes fondamentales :
1	ouvrir le fichier,
2	lire ou écrire dans ce fichier,
3	fermer ce fichier.

Navigation dans les répertoires (python / scilab)
Répertoire courant
>>> from os import getcwd
>>> getcwd()	--> pwd
Modifier le répertoire courant
>>> from os import chdir
>>> chdir('D:\\Test')	--> chdir('D:\Test\')

-	python : fonction open (plusieurs modes), méthodes read, write, close utilisables par l’objet représentant le fichier ouvert.
scilab : file (‘open ‘, ‘close’), write, read. 

Il existe plusieurs méthodes pour accéder à un fichier :
•	l’ouverture d’un fichier en lecture seule (r) permet d’ouvrir le fichier sans le modifier ;
•	l’ouverture d’un fichier en écriture (w) permet d’ouvrir un fichier. Attention, si le fichier existe il est écrasé ;
•	l’ouverture d’un fichier en mode ajout (a) permet d’ajouter des informations à un fichier existant déjà.
Bonnes pratiques : savoir gérer efficacement un ensemble de fichiers correspondant à des versions successives d’un fichier source. 

Exemple avec python :
>>> a = open("test.txt", "r")		#ouverture en lecture en mode texte
>>> a.read(5)			#lecture des cinq premiers caractères
'toto '
>>> a.close()			#fermeture
>>> a = open("image1.bmp", "rb")	#ouverture en lecture en mode binaire
>>> a.seek(5)			#va au 6ème byte
>>> a.read(4)			#lit 4 bytes
b'\x00\x04\x06\x00'
>>> a.close()			#ferme le fichier
